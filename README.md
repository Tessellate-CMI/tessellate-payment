## Developing

1. Install [bun](https://bun.sh/docs/installation)
2. Clone the repository

```sh
git clone https://github.com/Tessellate-CMI/tessellate-frontend
```

3. Install dependencies

```sh
cd tessellate-payment
bun install
```

4. Start the server

```sh
bun run dev
```
